# utility - keep text before last or 1st comma (get "Waco" from "Waco, TX") not used

utility - keep text before last or 1st comma (get "Waco" from "Waco,
TX") not used

## Usage

``` r
pre_comma(x, lastcomma = TRUE, if_no_comma_do_nothing = TRUE, trim = FALSE)
```

## Arguments

- x:

  string vector

- lastcomma:

  logical

- if_no_comma_do_nothing:

  logical

- trim:

  logical

## Value

vector like x
