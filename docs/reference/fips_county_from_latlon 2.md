# Find what county is where each point is located

Find what county is where each point is located

## Usage

``` r
fips_county_from_latlon(sitepoints = NULL, lat = NULL, lon = NULL)
```

## Arguments

- sitepoints:

  data.frame with lat,lon columns

- lat:

  latitudes vector if sitepoints not provided

- lon:

  longitudes vector if sitepoints not provided

## Value

just vector of fips
