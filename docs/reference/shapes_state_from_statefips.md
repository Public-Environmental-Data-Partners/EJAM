# Get boundaries of State(s) for mapping

Get boundaries of State(s) for mapping

## Usage

``` r
shapes_state_from_statefips(fips)
```

## Arguments

- fips:

  vector of one or more State FIPS codes

## Value

spatial data.frame of boundaries
