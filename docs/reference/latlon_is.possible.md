# Check if lat lon between -180 and +180

Check if lat lon between -180 and +180

## Usage

``` r
latlon_is.possible(lat, lon)
```

## Arguments

- lat:

  vector of latitudes

- lon:

  vector of longitudes

## Value

logical vector, one element per lat lon pair (location)
