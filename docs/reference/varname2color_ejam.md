# helper function - for color coding excel sheet columns

Convert R variable name of indicator to appropriate color for header row
in Excel

## Usage

``` r
varname2color_ejam(varname, varnameinfo)
```

## Arguments

- varname:

  things like us.avg.pctlowinc

- varnameinfo:

  must be left as default currently

## Value

vector of colors
