# formulas_ejscreen_acs_disability dataset

formulas for calculating count and percent with disability, from ACS raw
data

## Usage

``` r
formulas_ejscreen_acs_disability
```

## Format

An object of class `data.frame` with 3 rows and 4 columns.

## Details

Used for annual update of ACS-based indicators related to population
with disabilities. See
[`calc_blockgroupstats_from_tract_data()`](https://public-environmental-data-partners.github.io/EJAM/reference/calc_blockgroupstats_from_tract_data.md)
and
[`calc_blockgroupstats_acs()`](https://public-environmental-data-partners.github.io/EJAM/reference/calc_blockgroupstats_acs.md)
