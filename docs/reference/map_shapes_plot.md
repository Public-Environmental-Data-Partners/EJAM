# Map - polygons - Use base R plot() to map polygons

Map - polygons - Use base R plot() to map polygons

## Usage

``` r
map_shapes_plot(shapes, main = "Selected Census Units", ...)
```

## Arguments

- shapes:

  like from
  shapes_counties_from_countyfips(fips_counties_from_state_abbrev("DE"))

- main:

  title for map

- ...:

  passed to plot()

## Value

Just draws map using plot()
