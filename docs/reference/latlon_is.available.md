# Check if lat lon not NA using !is.na()

Check if lat lon not NA using !is.na()

## Usage

``` r
latlon_is.available(lat, lon)
```

## Arguments

- lat:

  vector of latitudes

- lon:

  vector of longitudes

## Value

logical vector, one element per lat lon pair (location)
