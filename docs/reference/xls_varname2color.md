helper function - for color coding excel sheet columns - NOT USED (but

## Usage

``` r
xls_varname2color(varname, varnameinfo)
```

## Arguments

- varname:

  things like avg.pctlowinc

- varnameinfo:

  data.frame with info on type of each variable

## Value

vector of colors
