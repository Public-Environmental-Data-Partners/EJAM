# check if FIPS code is valid, meaning it is an actual Census FIPS code for a State, County, City/CDP, etc.

check if FIPS code is valid, meaning it is an actual Census FIPS code
for a State, County, City/CDP, etc.

## Usage

``` r
fips_valid(fips)
```

## Arguments

- fips:

  vector of numeric or character fips. works for state, county,
  city/cdp, tract, blockgroup or block fips.

## Value

logical vector
