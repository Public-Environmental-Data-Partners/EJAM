# estimate how long it will take to get buffer batch results

estimate how long it will take to get buffer batch results

## Usage

``` r
speedmessage(n, perhourslow = 1000, perhourfast = 12000, perhourguess = 6000)
```

## Arguments

- n:

  number of points to buffer at

- perhourslow:

  n per hour if slow (conservative estimate of time needed)

- perhourfast:

  n per hour if fast

- perhourguess:

  n per hour best guess
