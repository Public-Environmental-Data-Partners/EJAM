# sictable (DATA) data.table of SIC code(s) and industry names for each EPA-regulated site

data.table of SIC code(s) and industry names for each EPA-regulated site
in Facility Registry Service Also has the 2,3, and 4-digit SIC that this
code falls under, where relevant for given length

## Usage

``` r
sictable
```

## Format

An object of class `data.table` (inherits from `data.frame`) with 1117
rows and 6 columns.

## Details

This is similar to the data file EJAM::SIC but in a more useful format
and newer functions work with it. Also see [SIC
info](https://siccode.com)
