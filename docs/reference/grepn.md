# utility - how many times within each element of x vector is the pattern matched ?

utility - how many times within each element of x vector is the pattern
matched ?

## Usage

``` r
grepn(pattern, x)
```

## Arguments

- pattern:

  passed to [`gregexec()`](https://rdrr.io/r/base/grep.html)

- x:

  a vector of character strings

## Value

vector of numbers
