# See which lists of names the given indicator names are in

See which lists of names the given indicator names are in

## Usage

``` r
names_whichlist_multi(x, ...)
```

## Arguments

- x:

  vector of names (query terms)

- ...:

  passed to names_whichlist()

## Value

a list of sets of names
