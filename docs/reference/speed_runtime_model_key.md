# Build a runtime model lookup key

Build a runtime model lookup key

## Usage

``` r
speed_runtime_model_key(analysis_type, analysis_subtype = NULL)
```

## Arguments

- analysis_type:

  input mode such as `"points"`, `"fips"`, or `"shapefile"`.

- analysis_subtype:

  optional subtype such as `"city"` or `"county"`.

## Value

Runtime model lookup key.
