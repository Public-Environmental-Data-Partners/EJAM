# censusplaces (DATA) Census FIPS and other basic info on roughly 40,000 cities/towns/places

Table of US cities and other Census Designated Places, Incorporated
Places, and County Subdivisions

## Usage

``` r
censusplaces
```

## Format

An object of class `data.frame` with 38195 rows and 7 columns.

## Details

from <https://www2.census.gov/geo/docs/reference/codes/PLACElist.txt>
Column names: eparegion, ST, stfips, countyname, countyfips, placename,
fips
