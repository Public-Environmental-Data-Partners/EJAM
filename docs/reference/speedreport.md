# helper function that reports on how long an analysis took

helper function that reports on how long an analysis took

## Usage

``` r
speedreport(start, end, n)
```

## Arguments

- start:

  start time

- end:

  end time

- n:

  how many places were analyzed

## Value

text string summarizing the speed
