# Helper function to view popup info in an interactive session - easier format to view

Helper function to view popup info in an interactive session - easier
format to view

## Usage

``` r
popup_print(x, linkregex = "<a href.*>(.*)<.*", linksimple = "\\1")
```

## Arguments

- x:

  output of
  [`popup_from_ejscreen()`](https://ejanalysis.github.io/EJAM/reference/popup_from_ejscreen.md)

- linkregex:

  see source

  see source
