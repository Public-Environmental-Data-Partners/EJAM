# islandareas (DATA) table, bounds info on lat lon of US Island Areas

data.frame of info on approximate lat lon bounding boxes around American
Samoa, Guam, the Commonwealth of the Northern Mariana Islands (Northern
Mariana Islands), and the United States Virgin Islands.

See also
[stateinfo](https://ejanalysis.github.io/EJAM/reference/stateinfo.md)
and
[stateinfo2](https://ejanalysis.github.io/EJAM/reference/stateinfo2.md)

See [Census
documentation](https://www.census.gov/programs-surveys/geography.html)

See source package files datacreate_islandareas.R or
EJAM/data-raw/islandareas.xlsx

Note the US minor outlying islands are not in that list and are widely
dispersed. They include Midway Islands, etc.

## Usage

``` r
islandareas
```

## Format

An object of class `data.frame` with 8 rows and 5 columns.
